使い方

初回は空の表が表示されています。
まず、記録したい株式の銘柄情報を登録します。
メニューのView -> Show Add BookInfo Window　を選択してください

SECURITIES CODE　に　証券コードを半角数字で入力してください
BOOK NAME にそのコードの株式名を記入してください。日本語も使えます
MARCKET は選択制になっています。略称になっていますが
TSE1東証1部
TSE2東証2部
JASDAQ東証ジャスダック
MATHERS東証マザーズ
OTHERその他、アメリカ株とか
で対応しています。 無事に登録できているかどうかは メニューのView -> Show BookInfo Window で確認できます。

次に取引記録をつけていきます。
メニューのView -> Show AddLog Window で確認できます。

Dateに日付を入力します。ボタンの横のカレンダーマークを押して出てきた カレンダーから選択してください。
Codeは証券コードを。登録したコードから選択します。
PurchasePrice は買値
PurchaseNum　は買い株数
SellingPrice　は売値
SellinNumは　は売り株数です。

一通り入力できましたら。Add to tableボタンを押してください。
テーブルに追加されましたら成功です。
